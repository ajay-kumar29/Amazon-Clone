This is a simple **Amazon Clone UI** built using **HTML, CSS, and JavaScript**.  
It replicates the look and feel of Amazon's homepage with features like navigation bars, product sections, sliders, and a responsive design.

---

## Features
- Top & bottom navigation bars  
- Sidebar menu with categories  
- Background image slider (auto & manual)  
- Product cards with sections  
- Product carousel sliders  
- Responsive layout  
- Footer Sections
